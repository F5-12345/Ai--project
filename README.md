# Ai--project 自媒体文章自动生成系统

## 项目需求
整体分为四大功能模块：文稿生成、AI配图、文件排版打包、自媒体平台自动发布。
系统架构采用 EXE -> API -> LLM 的层级结构，依靠调度器轮询控制任务执行，采用前序输出作为后续输入的方式，支持任务暂停、断点续跑，各模块独立日志隔离，方便问题回溯。

## 人员分工
1. 架构负责人：设计整体程序架构，编写任务调度主控逻辑，管理Git仓库，把控代码合并与开发流程。
2. 文稿开发：实现选题、文献检索、5000字正文生成与审稿校验，维护文稿模块运行日志。
3. 图文排版：对接AI绘图接口实现多轮对话配图，开发作者信息录入、文件自动打包为ZIP的功能。
4. 发布模块开发：实现平台筛选、账号自动注册、验证码识别、发布异常检测与返修功能。
5. 测试与文档管理：全流程测试排查问题，记录全部运行日志，完善项目文档，跟踪迭代进度。
   
## 开发流程
严格按照「文稿生成 → 配图绘图 → 排版打包 → 自媒体分发」的顺序增量开发；
使用Git做代码版本管理，每个模块拥有独立运行日志；
AI执行出现错误时，可通过日志反向定位提示词问题，模块间互不干扰。
